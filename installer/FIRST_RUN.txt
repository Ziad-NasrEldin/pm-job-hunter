PM Job Hunter - First Run Notes

1) The app opens in your browser on launch.
2) Local config file:
   %LOCALAPPDATA%\PMJobHunter\.env.local
3) Local data path:
   %LOCALAPPDATA%\PMJobHunter\data

Facebook setup:
1) Open the app and use "Facebook Login" in Quick Actions.
2) Complete login in the browser window until status says session is ready.
3) Run "Group Discovery", approve groups, then use "Run Facebook Scraper".

Note: this release is unsigned, so Windows SmartScreen may show a warning.
